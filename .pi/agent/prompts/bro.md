# Inspired by a Dillon Mulroy tweet (@dmmulroy).
description: Ask the agent to condense an overlong answer into a skimmable version
---
Bro — your last response was too much to read.

Please rewrite it into a much shorter, skimmable version without losing anything important.

Prioritize:
- The key takeaway first
- Only the important details I need to know
- Any warnings, risks, commands, file paths, decisions, or next steps
- Concise wording over exhaustive explanation

If you can preserve the same useful information in fewer words, do that. If some details are nonessential, omit them.
